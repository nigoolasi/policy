概述

咕咕拼音（App 显示名「拼音输入法」，键盘名「拼音键盘」）由个人开发者开发，致力于保护用户隐私。本隐私政策说明我们如何收集、使用和保护你的个人信息。

核心承诺：键盘扩展零权限、零网络、零数据收集。你输入的一切只存在于你的设备上。

Overview

Gugu Pinyin (App display name "拼音输入法 / Pinyin Input", keyboard name "拼音键盘 / Pinyin Keyboard") is developed by an individual developer and is committed to protecting your privacy. This Privacy Policy explains how we collect, use, and protect your personal information.

Our core promise: the keyboard extension requires zero permissions, zero network access, and collects zero data. Everything you type stays on your device.

我们收集什么数据

键盘扩展

✓ 不收集任何数据。

  • 不请求"允许完全访问"权限
  • 无法连接网络（iOS 系统强制隔离）
  • 不能读取通讯录、剪贴板、相册、位置
  • 不追踪输入内容，所有输入直接上屏到系统文本框
  • 本地词频学习完全在设备内完成，不上传

主应用

✓ 仅在以下场景使用网络：

  • 订阅与购买：连接 Apple App Store 验证和完成支付。此过程由 Apple 处理，我们无法访问你的支付信息。
  • 恢复购买：从 Apple 服务器同步已购项目。
  • 除此之外，主应用也不主动联网或上传任何数据。

What Data We Collect

Keyboard Extension

✓ We collect no data.

  • We do not request "Allow Full Access"
  • We cannot connect to the network (enforced by iOS sandbox isolation)
  • We cannot read your contacts, clipboard, photo library, or location
  • We do not track what you type — all input goes directly to the system text field
  • Local word-frequency learning happens entirely on-device and is never uploaded

Main App

✓ The network is used only in these scenarios:

  • Subscriptions & purchases: connecting to the Apple App Store to verify and complete payments. This is handled by Apple; we cannot access your payment information.
  • Restore Purchases: syncing previously purchased items from Apple's servers.
  • Apart from these, the main app does not proactively connect to the network or upload any data.

数据存储

以下数据仅存储在本地设备中：

  • 订阅状态：存储在 iOS Keychain 和 App Group 共享 UserDefaults 中，用于键盘扩展校验会员权限。
  • 用户词频 / 用户词库：你选字、选词的频率与个人词条记录，仅用于个性化候选排序。存储在 App Group 容器内，不上传。
  • 输入历史：不保存。每次键盘关闭即清除。

以上数据均不与任何第三方共享，不会离开你的设备。

Data Storage

The following data is stored only on your local device:

  • Subscription status: stored in iOS Keychain and the App Group shared UserDefaults, used only by the keyboard extension to verify membership.
  • User word frequency / user dictionary: records of the characters and words you choose, used only for personalized candidate ranking. Stored in the App Group container and never uploaded.
  • Input history: not saved; cleared every time the keyboard is dismissed.

None of this data is shared with any third party and never leaves your device.

本应用不使用任何分析 SDK、广告 SDK、统计工具或第三方追踪服务。

Third-Party Services

StoreKit is governed by the Apple Privacy Policy. We cannot access your Apple ID, payment method, or purchase details.

键盘扩展的隐私保护原理

iOS 键盘扩展运行在严格的系统沙箱中：

  • 默认情况下，键盘不能访问网络
  • 默认情况下，键盘不能访问其他 App 的数据
  • 只有用户主动在"设置 → 通用 → 键盘 → 拼音键盘"中开启"允许完全访问"，键盘才能获得有限的网络和数据库访问权限

咕咕拼音强烈建议用户关闭"允许完全访问"。本键盘所有功能（全键盘拼音、句级输入、用户词库、候选排序、主题、触感、音效）均在关闭完全访问的状态下正常工作。

需要说明：本键盘扩展在技术上声明了"完全访问"能力，但应用内不存在任何将输入内容或数据通过网络发送出去的代码；除非用户手动开启该权限，否则 iOS 的系统隔离会阻止键盘联网。

How the Keyboard Extension Protects Your Privacy

iOS keyboard extensions run in a strict system sandbox:

  • By default, the keyboard cannot access the network
  • By default, it cannot access data from other apps
  • Only if you manually enable "Allow Full Access" under Settings → General → Keyboard → Pinyin Keyboard does the keyboard gain limited network and database access

Gugu Pinyin strongly recommends keeping "Allow Full Access" turned off. All keyboard features (full-keyboard pinyin, sentence-level input, user dictionary, candidate ranking, themes, haptics, key sounds) work normally with Full Access disabled.

Note: technically this keyboard extension declares the "Full Access" capability, but the app contains no code that sends your input or data over the network; unless you manually enable it, iOS sandbox isolation prevents the keyboard from connecting to the network.
