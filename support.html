常见问题

  • 键盘无法弹出？ 前往 设置 → 通用 → 键盘 → 键盘 → 添加新键盘，选择"嘎嘎-简体笔画输入法"。
  • 需要开启"允许完全访问"吗？ 不需要，且强烈建议关闭。所有功能在关闭状态下均正常使用。
  • 订阅如何恢复？ 打开嘎嘎主应用 → 订阅页面 → 点击"恢复已购项目"。
  • 订阅如何取消？ iPhone 设置 → Apple ID → 订阅 → 找到嘎嘎 → 取消订阅。
