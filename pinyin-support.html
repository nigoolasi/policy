A. 功能与设置使用 FAQ

如何启用键盘？

前往 设置 → 通用 → 键盘 → 键盘 → 添加新键盘，选择「拼音键盘」，按提示完成添加。在任意可输入界面点击地球仪 🌐 图标即可切换到本键盘。首次使用无需任何特殊授权。

支持哪些输入方式？

  • 全键盘拼音：标准 QWERTY 全键盘，支持九键/全键布局。
  • 句级（整句）输入：基于 RIME 引擎的整句拼音输入，一次输入一整句话并智能分词。
  • 智能候选：候选词三级排序 + Bigram 语言模型预测，越用越顺手。

内置哪些词库？

  • 拼音词库：约 156 万词组，覆盖日常与专业用语。
  • 英文词库：常用英文单词。
  • 网络热词 / emoji 词典（sassy）：流行语与表情。
  • 诗词词频：常用诗词精选。

如何管理用户词库 / 记忆？

本键盘会学习你的选词习惯以优化候选排序，所有学习数据仅保存在本机。如需清空，请在主 App「设置 → 重置用户词库」中操作，一键清除全部个人学习数据，不影响系统其他内容。

主题与外观

  • 青色主色调，跟随系统亮 / 暗模式自动切换。
  • 玻璃模糊背景、圆角按键与细腻动画。

触感与音效

  • 触感反馈：提供关闭 / 轻 / 中 / 重 四档，可在设置中调节。
  • 按键音：会员功能会员，开通后可在设置中开关并调节音量。
  • 长按弹出表情 / 标点网格：年订阅专属年订阅，长按按键即可呼出自定义表情或标点面板。

A. Features & Setup FAQ

How do I enable the keyboard?

Go to Settings → General → Keyboard → Keyboards → Add New Keyboard, select "Pinyin Keyboard", and follow the prompts. Tap the 🌐 (globe) icon in any text field to switch to this keyboard. No special permission is required on first use.

What input methods are supported?

  • Full-keyboard Pinyin: standard QWERTY layout.
  • Sentence-level input: whole-sentence pinyin input powered by the RIME engine, with smart segmentation.
  • Smart candidates: three-tier candidate ranking + Bigram language-model prediction that learns as you type.

What dictionaries are built in?

  • Pinyin dictionary: ~1.56 million phrases covering everyday and professional usage.
  • English dictionary: common English words.
  • Slang / emoji dictionary (sassy): trending phrases and emojis.
  • Poetry word frequency: a curated selection of common poems.

How do I manage the user dictionary / memory?

The keyboard learns your word choices to improve candidate ranking; all learning data stays on your device. To clear it, open the main app and go to Settings → Reset User Dictionary to wipe all personal learning data in one tap — it does not affect anything else on your device.

Theme & appearance

  • Cyan accent color that follows the system Light / Dark mode automatically.
  • Glass blur background, rounded keys, and subtle animations.

Haptics & sound

  • Haptic feedback: Off / Light / Medium / Heavy, adjustable in Settings.
  • Key sounds: a membership featureMember — once unlocked, toggle and adjust volume in Settings.
  • Long-press emoji / punctuation grid: exclusive to yearly subscribersYearly — long-press a key to bring up a custom emoji or punctuation panel.

B. Membership, In-App Purchases & Refunds

Purchase models

  • Lifetime (one-time): a single payment that permanently unlocks all membership features.
  • Subscription: monthly or yearly auto-renewing plan that unlocks all membership features.
  • Yearly-exclusive: the yearly plan additionally includes exclusive capabilities such as the long-press emoji / punctuation grid.
  • Free trial: new users get a 24-hour free trial of all membership features (via StoreKit 2).

Plan differences

Lifetime / Subscription: key sounds, all themes and haptic levels, an ad-free clean experience.

Yearly-exclusive: long-press custom emoji / punctuation grid.

Free version: full pinyin input and dictionaries are usable; only some enhanced features require membership.

How do I restore purchases?

After switching devices or reinstalling, open the main app and tap "Restore Purchases" (based on AppStore.sync()) to sync your purchased items back to the current Apple ID — no need to pay again.

How do I request a refund?

All payments are handled by Apple; the developer cannot issue refunds directly. To request a refund, use Apple's official channel:

  • Visit reportaproblem.apple.com and sign in with the same Apple ID.
  • Find the relevant order, choose "Report a Problem / Request a Refund" and follow the prompts.
  • Refund decisions are made by Apple and are typically processed within 1–2 weeks.

D. 故障排查

键盘不显示 / 候选为空

  • 确认已在 设置 → 通用 → 键盘 → 键盘 中添加并开启「拼音键盘」。
  • 在输入界面点击地球仪 🌐 图标，确认已切换到本键盘。
  • 「允许完全访问」为可选功能，关闭状态下所有输入功能均正常。

为什么「无法联网 / 候选不更新」？

这是正常现象。键盘扩展默认处于系统离线隔离状态，所有输入与候选均在本地完成；联网仅发生在你主动打开购买页面时。无需担心，也不是故障。

会员 / 订阅不生效

  • 打开主 App，点击「恢复购买」重新同步。
  • 确认当前 Apple ID 与购买时一致。
  • 尝试重启设备或在设置中移除并重新添加键盘。

用户词库异常 / 候选变差

在「设置 → 重置用户词库」中清空学习数据即可恢复初始状态（不会删除系统词库）。

重装后免费试用被重置？

试用起点记录在 iOS Keychain 中以防重装滥用，因此重装应用后试用计时可能重新计算。如已付费，请使用「恢复购买」同步会员状态。

D. Troubleshooting

The keyboard doesn't appear / candidates are empty

  • Make sure "Pinyin Keyboard" has been added and enabled under Settings → General → Keyboard → Keyboards.
  • In a text field, tap the 🌐 (globe) icon to confirm you've switched to this keyboard.
  • "Allow Full Access" is optional; all input features work normally with it off.

Why "can't connect to the network / candidates don't update"?

This is normal. The keyboard extension is offline by default under iOS sandbox isolation; all input and candidates are processed locally. The network is only used when you proactively open the purchase page. There is nothing wrong, and no need to worry.

Membership / subscription not working

  • Open the main app and tap "Restore Purchases" to re-sync.
  • Confirm the current Apple ID matches the one used for purchase.
  • Try restarting the device, or remove and re-add the keyboard in Settings.

User dictionary abnormal / worse candidates

Go to "Settings → Reset User Dictionary" to clear learning data and return to the initial state (the built-in system dictionaries are not deleted).

Free trial reset after reinstall?

The trial start time is stored in iOS Keychain to prevent abuse via reinstalling, so the trial timer may restart after reinstalling the app. If you've already paid, use "Restore Purchases" to sync your membership status.
